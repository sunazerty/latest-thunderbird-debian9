# latest-thunderbird-debian9

You will be able to install Thunderbird 52.2.1-4 on Debian 9 (stretch)

## Clone the repository

First, you have to install Git with this command:
```
sudo apt-get update && sudo apt-get -y install git
```
Then, you can clone the repository with:
```
git clone https://github.com/sunazerty/latest-thunderbird-debian9.git

### Installing

In a shell:
```
cd latest-thunderbird-debian9
```
```
chmod +x setup.sh
```
```
sudo setup.sh -i YOUR_ARCH
```
